HptiBine custom color PNG compatibility set

The uploaded HptiBine 1.2.5 HD C6 source archive contains Java source only and no PNG assets to copy.
These PNGs are generated default-compatible sample maps for the HptiBine-supported paths that HptiBine now loads from texture packs:

  misc/grasscolor.png
  misc/foliagecolor.png
  misc/watercolorX.png
  misc/pinecolor.png
  misc/birchcolor.png
  misc/swampgrasscolor.png
  misc/swampfoliagecolor.png

To use them in-game, copy this misc folder into a texture pack and enable Quality -> Custom Colors.
They are included as source/package compatibility helpers, not as original HptiBine artwork.
